Sublime Text 之 Package Control

5月 14 2014

Package Control 是 ST 的插件（别人都叫 plugin，它叫 package）管理系统，管理其他 package 的。

ST package 的安装目录（查看已安装的package）在：
Preferences->Browse Packages->Installed Packages

开始使用：ctrl+shift+p -> Package Control:

  • Install package：安装 package，从官方默认channel获取列表
  • Remove Package：
    • 会删掉 installed packages目录下的package
    • 修改 Packages/User/Package Control.sublime-settings 文件中的 package
  • Add Repository：安装channel中尚没有的package，需给出一个git库目录
  • Upgrade Package：列出可供升级的package
  • Upgrade/Overwrite All Packages：升级并覆盖所有package，包括不是通过package control安装的
